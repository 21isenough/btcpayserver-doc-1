# BTCPay Magento Integration

If you're using Magento, an open-source e-commerce platform written in PHP, you can integrate BTCPay Server and use it as a payment processor.

# Magento 1

  1. Download BTCPay Magento Plugin (opens new window)
  2. Follow the quick-start guide here (opens new window)

# Magento 2

We recommend using the Magento 2 module developed by Storefront.be (opens new window), since it is the most feature-complete and robust. Storefront is also committed to maintaining the module as newer Magento versions are released.

The Magento 2 module is available for free at https://github.com/storefront-bvba/magento2-btcpay-module
